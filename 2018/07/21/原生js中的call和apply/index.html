原生js中的call和apply

Posted on 2018-07-21 |

obj.call(thisObj, arg1, arg2, …);
obj.apply(thisObj, [arg1, arg2, …]);
两者作用一致，都是把obj(即this)绑定到thisObj，这时候thisObj具备了obj的属性和方法。或者说thisObj『继承』了obj的属性和方法。绑定后会立即执行函数。

唯一区别是apply接受的是数组参数，call接受的是连续参数。

function add(j, k){
return j+k;
}

function sub(j, k){
return j-k;
}
我们在控制台运行：

add(5,3); //8
add.call(sub, 5, 3); //8
add.apply(sub, [5, 3]); //8

sub(5, 3); //2
sub.call(add, 5, 3); //2
sub.apply(add, [5, 3]); //2
